# Документация для сайта

Эта документация предоставляет описание различных секций и информации, содержащейся на веб-сайте. Вот обзор основных секций:

## Заголовок
Сайт содержит заголовок "Меня зовут Рияз Вагапов", который представляет имя и фамилию веб-разработчика.

## Социальные сети
В разделе "Социальные сети" представлены иконки популярных социальных сетей, таких как GitHub, Instagram и Facebook. Эти иконки предоставляют ссылки на профили веб-разработчика в социальных сетях.

## Основная информация
Этот раздел содержит основную информацию о веб-разработчике. Включает в себя фотографию, полное имя и заголовок "Я web-разработчик из города Казань". Также есть две кнопки "Портфолио" и "Обо мне", которые предлагают переход к соответствующим разделам на странице.

## Меню
Меню содержит список пунктов навигации, которые переносят пользователя к различным секциям на странице. Пункты меню включают "ПРО МЕНЯ", "МОЙ ОПЫТ", "МОИ НАВЫКИ", "МОИ РАБОТЫ", "ПРАЙС-ЛИСТ" и "КОНТАКТЫ".

## Обо мне
Раздел "Про меня" предоставляет информацию о веб-разработчике. Включает фотографию, заголовок "Про меня", описание, а также два блока с информацией о навыках веб-разработки и дизайне.

## Галерея
Галерея содержит несколько фотографий, которые представляют разные моменты из жизни веб-разработчика. Пользователь может просматривать фотографии, используя стрелки "влево" и "вправо".

## Опыт
Раздел "Опыт" содержит информацию о образовании и опыте веб-разработчика. Включает заголовок "Опыт", информацию о высшем образовании в Казанском федеральном университете и курсы по веб-разработке на платформе Udemy.

## Навыки
Раздел "Навыки" представляет список навыков, которыми обладает веб-разработчик. Включает языки программирования и технологии, такие как HTML5, CSS3, JavaScript, Node.js, PostgreSQL и jQuery. Также показывает уровень владения каждым навыком в процентах.

## Портфолио
Раздел "Портфолио" содержит примеры работ веб-разработчика. Каждая работа представлена в виде ссылки, которая перенаправляет пользователя на соответствующую страницу с подробностями работы.

## Прайс-лист
Раздел "Прайс-лист" показывает информацию о стоимости различных услуг веб-разработчика. Включает три категории услуг: "Landing page", "Многостраничный сайт" и "Интернет-магазин".

## Контакты
Раздел "Контакты" предоставляет информацию о различных способах связи с веб-разработчиком. Включает ссылки на социальные сети и форму для отправки сообщения, где пользователь может указать свое имя, электронную почту и сообщение.

Это общее описание структуры и содержимого веб-сайта. Каждая секция представляет собой отдельный блок информации и функциональности.
